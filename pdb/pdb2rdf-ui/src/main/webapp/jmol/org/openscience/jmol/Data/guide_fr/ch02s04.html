-Xmx

Avec cette option vous pouvez changer la taille maximum du tas que la machine virtuelle Java peut adresser dans les systèmes Java 1.4. Par défaut, elle est de 64 Mo. Pour de gros fichiers ce peut être insuffisant. En utilisant

[-Xmx512m]

la taille du tas est agrandie à un demi gigaoctet. Sur des systèmes Java plus anciens la syntaxe de cette option est différente (cf. java --help).
