> Pruebas locales (sin servidor)

Prueba en el ordenador local de páginas web creadas con el “Generador de páginas web de Jmol”

(No funciona para la plantilla ScriptButton si usas Firefox bajo MacOSX, pero sí funciona con Safari en MacOSX y con Firefox en Linux y Windows.  Al parecer, Firefox-Mac pasa rutas absolutas en lugar de relativas.)

El código JavaScript incluido en las páginas comprueba si la página se ha cargado desde un servidor o desde el ordenador local.  En este segundo caso, se usa la ruta relativa a Jmol que se indicó en la caja de texto “ruta local”. La carpeta o directorio que contiene la página que se está probando debe estar contenida (ser un subdirectorio) en la que contiene la miniaplicación Jmol en el ordenador local. Por lo tanto, para probar las páginas creadas con el Generador de páginas web de Jmol en tu propio ordenador, debes seguir estos pasos:

  1. Crea una carpeta (o directorio) con un nombre como “pruebas” dentro de la carpeta de tu disco local que contenga la miniaplicación Jmol.
  2. Indica “../../” como ruta local a Jmol cuando crees la página en el generador de páginas web de Jmol.
  3. Copia o traslada la carpeta (no sólo su contenido) que contiene los archivos .html, .spt, .png y .js de tu página a la carpeta que creaste en el paso 1.
  4. Abre el archivo .html en el navegador que quieras usar para las pruebas. La página debe funcionar igual que lo hará en un servidor.
