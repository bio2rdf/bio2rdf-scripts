-Duser.language

En plus des options ci-dessus, il y a une option Java importante:

[-Duser.language]

. Avec cette option la langue de l'IHM peut être choisie. Actuellement, les langages suivant sont supportés: de, en, es, fr, nl et pl.

Alternativement, vous pouvez aussi utiliser la variable d'environnement LANG pour choisir la langue préférée. C'est habituel dans beaucoup de systèmes UNIX, comme Linux. Toutefois, il est connu que celà ne marche pas sous Solaris et l'option ci-dessus de la ligne de commande doit être utilisée.
