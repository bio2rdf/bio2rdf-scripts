Le menu Affichage

Dans ce menu les réglages/styles d'affichage peuvent être modifiés.

En général, les styles d'affichage sont appliqués au groupe d'atomes et de liens actuellement sélectionnés.

Par défaut, tous les atomes et les liens dans une molécule sont sélectionnés.

Vous pouvez sélectionner des atomes/liens individuels en utilisant le curseur de sélection ou en utilisant des commandes de script.
