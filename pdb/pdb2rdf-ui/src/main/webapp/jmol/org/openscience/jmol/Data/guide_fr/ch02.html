Le programme Jmol a une option de la ligne de commander pour exécuter des scripts:

[--script]

. La syntaxe courante est:

jmol [--script {rasmol.script}]

Ceci permet de lancer jmol avec un script Rasmol. La syntaxe de ces scripts est expliquée dans Scripts RasMol/Chime.

Alternativement, Jmol peut être lancé avec un argument étant un nom de fichier:

jmol {file}

--geometry

Cette option permet de spécifier la taille de la fenêtre. Par défaut, elle est de 500x500.
