The File menu

Open

Open files in Jmol. Jmol can read a large number of file types, including:

  • ABINIT

  • ACES II

  • ADF - Amsterdam Density Functional

  • CML v2.0

  • GAMESS

  • Gaussian 92/94/98

  • Ghemical MM

  • Jaguar

  • MDL molfile

  • PDB

  • CIF/mmCIF

  • XYZ (single and multiple frame)

Recent Files...

Brings up a dialog with a list of the files the program has been asked to open recently. The file can be opened again by selecting it and clicking the Open button.

Script...

Opens a window where Jmol/RasMol/Chime scripting commands can be executed. RasMol/Chime Scripts contains information about RasMol/Chime Scripts.

Export

Export -> Export Image...

Jmol can export 24 bit color images of the following types:

  • JPG

  • PNG

  • PPM

Export -> Render in pov-ray...

For high quality images, Jmol interfaces with the POV-Ray ray tracer. To use this feature, you must have the povray command line utility installed. You can download it from http://www.povray.org/.

Print

Postscript files can be exported directly to printer and to file.

Exit

Quits this Jmol session.
